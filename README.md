## Overview

This project visualizes the return on investment (ROI) of higher education by combining storytelling, interactive maps, and custom visualizations. Users navigate an ASU-inspired campus map and encounter data at each location, comparing the financial benefits and risks of pursuing a college degree.

The experience is designed to let users form their own conclusions based on evidence rather than opinion.
